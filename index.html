About This Project

The Weather Dashboard is developed as part of the CodTech Internship. It demonstrates integration with a public API (Open-Meteo) to fetch real-time weather data and display it dynamically in a responsive user interface.

Features include:

  • City-based weather search
  • Current temperature, wind, and pressure
  • 12-hour forecast display
  • Detailed 24-hour table with humidity and wind

Built with: HTML, CSS, and JavaScript.
